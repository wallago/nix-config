# Zen - Browser

## Compact Mode

| Action                  | Key          |
| ----------------------- | ------------ |
| Toggle Floating Sidebar | `Ctrl+Alt+X` |
| Toggle Compact Mode     | `Ctrl+Alt+C` |

## Workspaces

| Action                  | Key      |
| ----------------------- | -------- |
| Close All Unpinned Tabs | `Alt+C`  |
| Forward Workspace       | `Ctrl+O` |
| Backward Workspace      | `Ctrl+N` |

## URL

| Action                       | Key            |
| ---------------------------- | -------------- |
| Copy current URL as Markdown | `Ctrl+M`       |
| Copy current URL             | `Ctrl+Shift+C` |

## Tabs

| Action                  | Key            |
| ----------------------- | -------------- |
| New Tab                 | `Ctrl+T`       |
| Close Tab               | `Alt+Q`        |
| Restore Last Closed Tab | `Ctrl+Shift+T` |
| Select Tab              | `Alt+[1-9]`    |

## Navigation

| Action                   | Key            |
| ------------------------ | -------------- |
| Go Forward               | `Alt+O`        |
| Go Backward              | `Alt+N`        |
| Reload Page              | `Ctrl+R`       |
| Reload Page (skip cache) | `Ctrl+Shift+R` |

## Search & Find

| Action        | Key            |
| ------------- | -------------- |
| Focus Search  | `Ctrl+K`       |
| Find on Page  | `Ctrl+F`       |
| Find Again    | `Ctrl+H`       |
| Find Previous | `Ctrl+Shift+H` |
