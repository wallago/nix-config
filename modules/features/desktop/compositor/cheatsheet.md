# Niri - Compositor

## General

| Action     | Key           |
| ---------- | ------------- |
| Overview   | `Mod+Tab`     |
| Close      | `Mod+Q`       |
| Exit       | `Mod+Shift+Q` |
| Screenshot | `Mod+P`       |
| Cheatsheet | `Mod+Shift+/` |

## Apps

| Action   | Key          |
| -------- | ------------ |
| Terminal | `Mod+Return` |
| Menu     | `Mod+D`      |
| Lock     | `Mod+Escape` |
| Browser  | `Mod+B`      |

## Layout

| Action                       | Key           |
| ---------------------------- | ------------- |
| Expand (Maximize Column)     | `Mod+F`       |
| Fullscreen                   | `Mod+Shift+F` |
| Rotate Width                 | `Mod+R`       |
| Center Column                | `Mod+C`       |
| Toggle Floating              | `Mod+V`       |
| Toggle Focus Floating/Tiling | `Mod+Shift+V` |
| Consume Window Left          | `Mod+[`       |
| Expel Window Right           | `Mod+]`       |

## Focus

| Action | Key     |
| ------ | ------- |
| Left   | `Mod+N` |
| Right  | `Mod+O` |
| Up     | `Mod+I` |
| Down   | `Mod+E` |

## Move Window

| Action | Key           |
| ------ | ------------- |
| Left   | `Mod+Shift+N` |
| Right  | `Mod+Shift+O` |
| Up     | `Mod+Shift+I` |
| Down   | `Mod+Shift+E` |

## Workspaces

| Action           | Key           |
| ---------------- | ------------- |

| Move Column Up   | `Mod+Shift+Y` |
| Move Column Down | `Mod+Shift+U` |
